Класс сотрудника:
 
 - No- Указывает номер сотрудника, первые две буквы отдела, к которому добавляются номера сотрудников
        и состоит из порядка создания произведения, причем порядок создания произведения начинается с 1000,
        например, номер вакансии, которая работает в отделе затрат и добавляется в систему как 35-я, должен быть MA1035
 
 - Fullname - имя и фамилия сотрудника
 - Position- должность сотрудника (должно содержать не менее 2 букв)
 - Salary - оклад работника не может быть ниже 250
 - DepartmentName - название отдела, в который добавляется вакансия

------------------------------------------------------------------------------------
Класс отдела:
 
 - Name - Указывает название отдела, отдел должен состоять не менее чем из 2-х букв
 - WorkerLimit - Указывает максимальное количество работников, которое может существовать в отделе, минимальное может быть 1
 - SalaryLimit - относится к максимальной месячной зарплате для всех должностей в отделе, минимальная может быть 250
 - Employees - Отображает сотрудников в отделе Отображает список сотрудников добавленных в отдел.
 - CalcSalaryAverage() - метод, который возвращает среднюю зарплату сотрудников отдела

IHumanResourceManager

 - Departments - относится к списку отделов в системе.
 - AddDepartment() - получает и создает в качестве параметра необходимую информацию для создания отдела,
                     метод, который добавляет в массив отделов
 - GetDepartments() - возвращает все отделы в системе
 - EditDepartments() - изменение в  отделе - можно изменить название отдела и параметр, который вы хотите изменить
                        отправляет название отдела и новое название

 - AddEmployee() - в качестве параметров передается необходимая информация и departament name для создания сотрудника, в методе создается сотрудник
                   и отправленный номер добавляется в массив Сотрудники отдела

- RemoveEmployee() - номер сотрудника и отдел обычно передаются в качестве параметров и удаляет сотрудника с отправленным номером из Сотрудников отдела

- EditEmployee() - параметры: DepartmentName, номер, ФИО, оклад и должность
